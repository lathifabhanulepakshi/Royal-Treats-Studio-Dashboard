# Royal-Treats-Studio-Dashboard
Royal Treats Studio – Ice Cream Sales Dashboard is an interactive business intelligence dashboard designed to analyze and visualize ice cream sales performance. The dashboard provides key insights into total sales, year-over-year growth, product performance, payment methods, and monthly trends.
#Royal Treats Studio – Business Insights Dashboard
An interactive Power BI dashboard designed for a dessert/ice-cream business to analyze sales performance, product trends, and overall business health using dynamic visuals, KPIs, slicers, and DAX measures.
## Project Overview
Royal Treats Studio is a Business Intelligence dashboard built in Microsoft Power BI that converts raw sales data into meaningful insights for a dessert or ice-cream business.
The dashboard helps stakeholders:
- Monitor revenue performance  
- Track product/category trends  
- Evaluate business growth  
- Make data-driven decisions  
This project demonstrates end-to-end BI development including data preparation, modeling, and interactive visualization.
## Business Problem
Food and dessert businesses generate large amounts of sales data daily. Without proper analytics, it becomes difficult to:
- Track total sales performance  
- Identify top-selling products  
- Monitor revenue trends over time  
- Understand category contribution  
- Make informed business decisions  
The Royal Treats Studio dashboard solves these challenges through an interactive reporting solution.
## Dashboard Features
✔ KPI Cards (Total Revenue, Total Orders, Key Metrics)  
✔ Product/Category Performance Analysis  
✔ Revenue Trend Over Time (Line Chart)  
✔ Sales Distribution (Pie/Donut Chart)  
✔ Dynamic Slicers for filtering  
✔ Interactive cross-filtering visuals  
✔ Custom DAX Measures  
✔ Clean and modern dashboard design  
## Tools & Technologies Used
- Microsoft Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Power Query Editor  
- Excel Dataset  
- Data Modeling & Relationships  
## DAX Measures Implemented
Example measures used in this project:
Total Revenue = SUM(Sales[Revenue])
Total Orders = COUNT(Sales[Order_ID])
